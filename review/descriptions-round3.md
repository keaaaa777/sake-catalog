## るみ子の酒 あるがまま 生酛純米(醇酒)

- 蔵元: 森喜酒造場(三重県)
- 特定名称: 生酛純米
- スペック: 精米歩合88% / ABV不明%
- slug: mie-rumikonosake-arugamama-kimoto

三重県伊賀市の森喜酒造場が、生酛造りにこだわって醸す純米酒です。精米歩合88%とほとんど米を削らずに仕込み、40日かけてじっくりと育てた酒母を低温でゆっくり発酵させています。米を大きく削らないことで生まれる骨太な旨みと、生酛由来のしっかりとした酸味が特徴です。無濾過生原酒として瓶詰めされることが多く、力強い飲みごたえがあります。常温からぬる燗にすると、より一層旨みが広がります。脂の乗った料理との相性が良い一本です。

## 花垣 年輪譜 大吟醸十年古酒(熟酒)

- 蔵元: 南部酒造場(福井県)
- 特定名称: 大吟醸
- スペック: 精米歩合45% / ABV不明%
- slug: fukui-hanagaki-nenpufu-daiginjo-junen-koshu

福井県大野市の南部酒造場が、精米歩合45%まで磨いた大吟醸酒を10年間熟成させた古酒です。長期熟成古酒でありながら、大吟醸らしい華やかな香りが失われずに残っているのが特徴で、鋭さのある酸味が全体を引き締めています。熟成による丸みと吟醸香の華やかさが同居する、他の熟成古酒とは一線を画す味わいです。冷やでも常温でも楽しめ、温度による変化も楽しめます。特別な席の食後酒としておすすめの一本です。

## 田中六五 糸島産山田錦 純米酒(醇酒)

- 蔵元: 白糸酒造(福岡県)
- 特定名称: 純米酒
- スペック: 精米歩合65% / ABV不明%
- slug: fukuoka-tanakarokujugo-junmai

安政2年創業の白糸酒造が、山田錦の産地として知られる糸島の地元産山田錦を使って醸す純米酒です。精米歩合は銘柄名の由来にもなっている65%で統一されています。木製の槽に石の重しを使う「はね木搾り」という珍しい伝統製法で搾られており、きれいで丸みのある口当たりに、爽やかな酸が感じられるスムースな味わいが特徴です。冷やから常温まで幅広く楽しめ、地元糸島の魚介料理との相性を意識して造られた一本です。

## 杉錦 生もと純米90(醇酒)

- 蔵元: 杉井酒造(静岡県)
- 特定名称: 生もと純米
- スペック: 精米歩合90% / ABV不明%
- slug: shizuoka-suginishiki-kimoto-junmai90

天保9年創業の杉井酒造が、醸造量の8割以上を占める生酛・山廃造りで醸す純米酒です。精米歩合90%とほとんど米を削らずに仕込み、通常の1〜2倍の手間をかけた伝統製法で深みのある味わいを引き出しています。生酛らしいしっかりとした酸味と、削らない米ならではの力強い旨みが特徴です。常温からぬる燗にすると、複雑な味わいがより一層感じられます。濃い味付けの家庭料理や、発酵食品との相性が良い一本です。

## 手取川 山廃仕込 本醸造(醇酒)

- 蔵元: 株式会社吉田酒造店(石川県)
- 特定名称: 本醸造
- スペック: 精米歩合不明% / ABV不明%
- slug: ishikawa-tedorigawa-yamahai-honjozo

白山市の吉田酒造店が、伝統の山廃仕込みで醸す本醸造酒です。同じ蔵の純米大吟醸とは対照的に、山廃由来のしっかりとした酸味とコクのある味わいを楽しめる一本です。手取川の伏流水を仕込み水に使い、骨太でありながら飲み飽きしない味わいに仕上げています。常温からぬる燗にすると、山廃らしい旨みがより引き立ちます。日常の食卓に寄り添う、濃いめの味付けの料理との相性が良い酒です。

## 奥播磨 播秋 山廃純米生原酒(醇酒)

- 蔵元: 下村酒造店(兵庫県)
- 特定名称: 山廃純米
- スペック: 精米歩合55% / ABV不明%
- slug: hyogo-okuharima-banshu-yamahai-junmai-namagenshu

明治17年創業の下村酒造店が、播州産の兵庫夢錦を100%使用して醸す山廃純米の生原酒です。精米歩合55%まで磨いた米を山廃仕込みで丁寧に仕込み、加水調整をしない生原酒として瓶詰めしています。山廃由来のしっかりとした酸味と、生原酒ならではの力強い旨みが特徴です。純米酒のみを醸造する蔵らしい、米の個性を生かした濃醇な味わいです。常温からぬる燗にすると、旨みがより豊かに感じられます。秋の実りの料理との相性が良い一本です。

## 初孫 伝承生もと本醸造(醇酒)

- 蔵元: 東北銘醸(山形県)
- 特定名称: 本醸造
- スペック: 精米歩合60% / ABV15.5%
- slug: yamagata-hatsumago-densho-kimoto-honjozo

明治26年創業の東北銘醸が、全量生酛造りにこだわって醸す本醸造酒です。精米歩合60%の米を伝統の生酛造りで仕込み、日本酒度0・酸度1.5と穏やかな数値ながら、生酛由来の深みと旨みを感じられる味わいに仕上げています。燗にするとより一層旨みが引き立つのが特徴です。手頃な価格帯で、日常的に生酛造りの味わいを楽しめる一本です。家庭料理や焼き魚など、幅広い料理と合わせやすい酒です。

## 会津娘 本醸造にごり原酒(醇酒)

- 蔵元: 高橋庄作酒造店(福島県)
- 特定名称: 本醸造
- スペック: 精米歩合60% / ABV19%
- slug: fukushima-aizumusume-honjozo-nigori-genshu

会津若松市の高橋庄作酒造店が、地元の米と水にこだわって醸す本醸造のにごり原酒です。精米歩合60%の五百万石・夢の香を使い、加水調整をしない原酒として瓶詰めしているため、アルコール度数19%と力強い味わいになっています。もろみを粗く漉すことで生まれる濃厚な口当たりと、しっかりとした旨みが特徴です。常温からぬる燗にすると、まろやかさが増して飲みやすくなります。濃い味付けの鍋料理などとの相性が良い一本です。
