# ここに背景動画を配置してください

トップ画面(ファーストビュー)の背景に使う動画ファイルを、このフォルダに以下のファイル名で置いてください。

- `hero.mp4` — 背景動画本体(必須)
- `hero-poster.jpg` — 動画読み込み前/読み込み失敗時に表示する静止画(任意)

配置するだけで自動的に読み込まれます(コード変更は不要です)。

## 推奨仕様

- 長さ: 10〜20秒程度のループ再生を想定(無音)
- 解像度: 1920x1080 前後で十分(横長・スマホでは中央がクロップされます)
- 容量: できるだけ軽量に(5〜10MB程度を目安。重いとファーストビューの表示が遅くなります)
- 音声トラックは不要です(自動再生のためミュート再生になります)
